Collecting and Sourcing Images that are not yours

  • Use this Link to Search Creative Commons for images you can use
  • Type in a search for beaches
  • On the left side of the screen where you can filter the results, check out the licence or see if the image is in the public domain.
  • From the remaining results, choose an image. See how the author wants to be credited
  • Copy and paste the html indicated in the tab
